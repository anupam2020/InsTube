# InsTube
Download Instagram videos and Youtube videos/audios in one click with search in-app feature!


### Libraries :
* Lottie Animations
* RapidAPI
* OkHttp
* StatusBarUtil
* Glide
* Circle ImageView
* Dynamic Toast
